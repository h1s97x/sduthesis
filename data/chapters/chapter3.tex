\section{改进的轻量化模块设计}

\subsection{模块总体设计}

本文提出的改进轻量化模块（Enhanced Lite Block, ELB）在标准深度可分离卷积的基础上，引入了三个关键改进：

\begin{enumerate}
    \item \textbf{增强残差连接}：在残差路径中增加了一个$1 \times 1$的卷积层，用于调整通道维度，使残差连接更加高效。
    \item \textbf{通道注意力模块}：集成了轻量级的通道注意力机制，增强重要特征的表达能力。
    \item \textbf{多尺度特征融合}：通过并行使用不同感受野的卷积核，捕获多尺度的图像特征。
\end{enumerate}

\subsection{通道注意力实现}

本文采用一种轻量级的通道注意力模块，兼顾性能和计算效率。模块首先通过全局平均池化和全局最大池化分别提取通道的统计特征，然后使用轻量级MLP学习通道间的依赖关系。具体计算过程如算法\ref{algo:channel_attention}所示。

\begin{algorithm}[htbp]
    \caption{通道注意力模块计算流程}
    \label{algo:channel_attention}
    \begin{algorithmic}[1]
        \State \textbf{输入:} $X \in \mathbb{R}^{H \times W \times C}$
        \State \textbf{输出:} $X'$
        \State $z_{avg} = \text{GAP}(X)$ \Comment{全局平均池化}
        \State $z_{max} = \text{GMP}(X)$ \Comment{全局最大池化}
        \State $z = \text{Concat}(z_{avg}, z_{max})$ \Comment{特征拼接}
        \State $s = \sigma(\text{MLP}(z))$ \Comment{学习通道权重}
        \State $X' = s \odot X$ \Comment{通道加权}
        \State \Return \text{返回 } X'$
    \end{algorithmic}
\end{algorithm}

\section{数据预处理模块}

数据预处理是深度学习训练流程中的重要环节，合理的预处理策略能够显著提升模型的泛化能力。本文采用以下数据预处理方法：

\begin{itemize}
    \item \textbf{随机裁剪}：对输入图像进行随机尺寸和位置的裁剪，增强数据多样性。
    \item \textbf{随机水平翻转}：以50\%的概率对图像进行水平翻转。
    \item \textbf{颜色抖动}：随机调整图像的亮度、对比度、饱和度和色调。
    \item \textbf{归一化}：将图像像素值归一化到$[0, 1]$区间，再进行标准化处理。
\end{itemize}

\section{模型训练策略}

\subsection{损失函数}

网络采用交叉熵损失函数（Cross-Entropy Loss）作为训练目标，其表达式如式\eqref{eq:cross_entropy}所示：

\begin{equation}
    \mathcal{L}_{CE} = -\sum_{i=1}^{N} y_i \log(\hat{y}_i)
    \label{eq:cross_entropy}
\end{equation}

其中$N$为类别数，$y_i$为真实标签的概率分布，$\hat{y}_i$为模型预测的概率分布。

\subsection{优化器设置}

本文使用AdamW优化器进行模型训练，其特点是可以对权重衰减进行解耦，避免L2正则化与权重衰减的冲突。关键超参数设置如下：

\begin{itemize}
    \item \textbf{初始学习率}：0.001
    \item \textbf{权重衰减系数}：0.01
    \item \textbf{$\beta_1$, $\beta_2$}：0.9, 0.999
    \item \textbf{批次大小}：64（CIFAR-10）/ 256（ImageNet）
\end{itemize}

\subsection{学习率调度}

本文采用余弦退火学习率调度策略，学习率的变化如式\eqref{eq:cosine_lr}所示：

\begin{equation}
    \eta_t = \eta_{max} \cdot \frac{1}{2} \left(1 + \cos\left(\frac{t \pi}{T}\right)\right)
    \label{eq:cosine_lr}
\end{equation}

其中$\eta_{max}$为最大学习率，$T$为总训练轮数，$t$为当前训练轮数。
