\subsection{图片位置控制}

使用位置参数控制图片位置：

\begin{itemize}
  \item \texttt{h} - here，尽可能放在当前位置
  \item \texttt{t} - top，放在页面顶部
  \item \texttt{b} - bottom，放在页面底部
  \item \texttt{p} - page，单独放在一页
  \item \texttt{H} - 强制放在当前位置（需要 float 宏包）
\end{itemize}

\section{图片格式}

\subsection{支持的格式}

LaTeX 支持多种图片格式：

\begin{itemize}
  \item \textbf{PDF} - 矢量图，推荐用于图表、示意图（最佳）
  \item \textbf{PNG} - 位图，适合截图、照片
  \item \textbf{JPG/JPEG} - 位图，适合照片
  \item \textbf{EPS} - 矢量图（需要转换）
\end{itemize}

\subsection{图片路径}

图片路径可以是相对路径或绝对路径：

\begin{lstlisting}[language=TeX]
% 相对路径（推荐）
\includegraphics{assets/images/logos/sdu_logo_1.pdf}

% 也可以设置图片搜索路径
\graphicspath{{assets/images/}{figures/}}
\includegraphics{logos/sdu_logo_1.pdf}
\end{lstlisting}

\section{常见问题}

\subsection{图片不显示}

检查以下几点：
\begin{enumerate}
  \item 图片路径是否正确
  \item 图片格式是否支持
  \item 是否加载了 \texttt{graphicx} 宏包
  \item 编译器是否支持该格式（XeLaTeX 支持 PDF/PNG/JPG）
\end{enumerate}

\subsection{图片位置不对}

\begin{itemize}
  \item LaTeX 的浮动体机制会自动调整图片位置
  \item 使用 \texttt{[H]} 参数强制位置（需要 float 宏包）
  \item 调整浮动体参数：
\end{itemize}

\begin{lstlisting}[language=TeX]
\setcounter{topnumber}{2}      % 页面顶部最多2个浮动体
\setcounter{bottomnumber}{2}   % 页面底部最多2个浮动体
\renewcommand{\topfraction}{0.9}     % 顶部最多占90%
\renewcommand{\bottomfraction}{0.9}  % 底部最多占90%
\end{lstlisting}

\subsection{图片太大}

\begin{itemize}
  \item 使用 \texttt{width} 或 \texttt{height} 参数缩放
  \item 使用 \texttt{scale} 参数按比例缩放
  \item 使用 \texttt{max width} 和 \texttt{max height} 限制最大尺寸
\end{itemize}

\section{最佳实践}

\begin{enumerate}
  \item \textbf{使用矢量图}：图表、示意图使用 PDF 格式
  \item \textbf{合适的分辨率}：位图至少 300 DPI
  \item \textbf{统一风格}：保持图片风格一致
  \item \textbf{清晰的标题}：使用描述性的 caption
  \item \textbf{引用图片}：在正文中引用所有图片
  \item \textbf{合理命名}：使用有意义的 label
\end{enumerate}
